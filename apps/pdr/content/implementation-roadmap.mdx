## Phased Delivery

<Steps>
  <Step>
    ### Phase 1: Foundation (Q1 FY26)

    Stand up core infrastructure and migrate the first 3 data sources.

    - Provision Snowflake warehouse and configure networking
    - Deploy Lightdash on ECS with SSO integration
    - Migrate CRM, billing, and product usage data via Fivetran
    - Build 5 foundational dbt models (staging + marts)
    - Deliver MVP dashboard for Sales leadership

    **Exit criteria**: Sales team using self-service for weekly pipeline review

  </Step>
  <Step>
    ### Phase 2: Expand (Q2 FY26)

    Migrate remaining data sources and onboard additional teams.

    - Migrate remaining 9 data sources
    - Build 15 additional dbt models
    - Deploy scheduled reports for Finance and Marketing
    - Launch self-service training program
    - Implement RBAC for all departments

    **Exit criteria**: 50+ active users across 4 departments

  </Step>
  <Step>
    ### Phase 3: Optimize (Q3 FY26)

    Performance tuning, advanced features, and legacy sunset.

    - Convert all models to incremental refresh
    - Deploy real-time dashboard layer (WebSocket)
    - Build custom executive dashboard (Next.js)
    - Begin legacy tool deprecation
    - Conduct first cost optimization review

    **Exit criteria**: p95 load time < 3s, legacy tools deprecated

  </Step>
  <Step>
    ### Phase 4: Scale (Q4 FY26)

    Full rollout, documentation, and handoff.

    - Complete legacy tool sunset
    - Deploy to all 200+ business users
    - Finalize runbooks and operational documentation
    - Conduct post-implementation review
    - Transfer to BAU support team

    **Exit criteria**: Full platform handoff to operations

  </Step>
</Steps>
