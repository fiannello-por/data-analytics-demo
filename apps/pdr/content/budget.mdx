## 7.1 Paid Services

This architecture is built on open-source tools. The paid services required:

| Service                                                                                                                       | What it covers                                                                                 | Cost                                                                          |
| ----------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| **[Render](https://render.com/pricing)** (Professional)                                                                       | Hosting the Lightdash instance (Web Service $25 + PostgreSQL $19 + Professional workspace $19) | $63/month                                                                     |
| **[Cloudflare](https://www.cloudflare.com/plans/)** (Free)                                                                    | WAF, rate limiting, DDoS protection                                                            | Free                                                                          |
| **[GitHub Team](https://github.com/pricing)**                                                                                 | Repository, pull requests, code review, project management                                     | $4/user/month                                                                 |
| **[GitHub Actions](https://docs.github.com/en/billing/managing-billing-for-github-actions/about-billing-for-github-actions)** | CI/CD pipeline execution (validation, preview environments, deployment)                        | 3,000 minutes/month included with Team plan; $0.008/min after (Linux runners) |

## 7.2 Optional Services

| Service                                                | What it covers                                          | Cost                       |
| ------------------------------------------------------ | ------------------------------------------------------- | -------------------------- |
| **[OpenAI API](https://openai.com/pricing)**           | AI-powered PR review and automated changelog generation | Pay-per-use (per API call) |
| **[Anthropic API](https://www.anthropic.com/pricing)** | AI-powered PR review and automated changelog generation | Pay-per-use (per API call) |

Either provider can be used — the architecture is not locked to a specific LLM vendor.
